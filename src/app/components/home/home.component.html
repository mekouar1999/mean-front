Ce projet est un MEAN Stack Project :

  • Mongodb Base de donnée
  • Express Serveur
  • Angular User Interface Front-end
  • Node Js serveur


Ce Projet comporte :

  • Authentification via JSON Web Token
  • CRUD ( Assignement )
  • Pages sécurisé ( Auth Guard )
  • Identifiant et token stocké dans le local storage du client et valable pour une durée de 24h
  • Messages d'alert via des Flash messages ( succes , danger )


Déploiement du projet

  • L'app MEAN STACK sera déployé sur HEROKU via le lien suivant : ...
  • Pour lancer en local il suffira de :
  • Lancer le back-end depuis le repertoire api qui va lancer le back-end de l'app et ce sur le port 8010
  • Lancer le Front-end depuis le repertoire AssignementApp qui va lancer le Front-end de l'app et ce sur le port 4200
  • Changer l'URI de la BDD depuis server.js et config/database et mettre celle de votre BDD
